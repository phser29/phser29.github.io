# express

- 설치
> npm install express --save

> const express = require('express')

- EJS
  - 설치하면 특정 변수나 include를 사용가능

> npm i ejs

> const ejs = require('ejs');

## ejs 사용법

```
app.set('view engine'. 'ejs')
app.set('views', './views')


```

# anserCode

- 200 OK : 요청이 성공적으로 수행되었음
- 201 Created : 요청이 성공적으로 수행되었고, 새로운 리소스가 생성되었음
- 400 Bad Request : 요청 값이 잘못되어 요청에 실패했음
- 403 Forbidden : 권한이 없어 요청에 실패했음
- 404 Not Found : 요청 값으로 찾은 리소스가 없어 요청에 실패했음

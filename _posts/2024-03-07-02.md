# 유효성 체크

css의 display를 활용해서 ui에서 표시<br>

```
let idckCheck = true;
let idckCheck = false;

html : <span class="final_id_ck">아이디를 입력해주세요.</span>

css : final_id_ck{display: none;}

js : if(id == ""){
      $('.final_id_ck').css('display','block');
      idCheck = false;
  	}else{
      $('.final_id_ck').css('display', 'none');
      idCheck = true;
  	}

```

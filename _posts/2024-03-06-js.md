# javaScript 문법

## Variable

- var -> 이제는 안씀 <br>
- let <br>
- const <br>

```
var name = '코드 팩토리';

var age = 32;

let ive = '아이브';

const newJeans = '뉴진스';
```
- let과 var을 이용하면 값을 추후에 바꿀수 있음
- const는 이용하면 값을 바꿀수 없음
